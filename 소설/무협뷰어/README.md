# ⚔️ 무협지 뷰어

고전 무협지 전용 TTS 리더. 한자 자동 제거로 쾌적한 청취 가능.

## 특징
- 1,157권 무협지 지원 (한국/중국)
- TTS 재생 시 한자 자동 제거
- 작가별 필터, 제목/분량 정렬
- 북마크 / 이어읽기
- 화면 잠금 (이동 중 사용)
- 4가지 테마 (다크/라이트/나이트/그린)
- CP949/EUC-KR 인코딩 자동 감지

## 실행
```
START.bat
```
또는
```bash
python server.py
```
→ http://localhost:8906
